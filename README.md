# 🩺🤖 Medical Assistant

## 📌 Overview
Medical Assistant is an AI-powered application designed to provide reliable health-related information to users. It supports doctors, nurses, and individuals without a medical background by delivering clear, accessible answers to specific health questions. The goal is to make medical knowledge easier to understand and more widely available.

## 🎯 Features
- User Input Interface– Just type your question like you would in a conversation
- AI-Powered Responses – Get helpful, easy-to-understand answers
- Retrieval-Augmented Generation (RAG) – Pulls in relevant information to improve responses
- Built for Everyone – Designed for both medical professionals and the general public


## 🛠 Tech Stack
Python – Backend logic
OpenAI API – Powers the AI responses
LangChain – Handles the workflow between components
RAG (Retrieval-Augmented Generation) – Helps keep answers relevant and grounded
